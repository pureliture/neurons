확정 요약

  • RAGFlow는 neurons active/runtime 선택지에서 완전 퇴역한다.
  • 1차 범위는 neurons repo와 Mac local MCP surface다.
  • 보존 예외는 docs/retired/index-retirement.md 하나뿐이다.
  • 최종 repo-wide 검색 결과는 퇴역 기록 문서 외 0개를 목표로 한다.
  • 기존 index_* state/schema 이름은 backend-neutral 이름으로 migration한다.
  • 기존 데이터는 backup/snapshot 기반 보존 migration으로 옮긴다.
  • stale local MCP 프로세스와 설정은 승인 후 정리한다.
  • 실제 종료, 수정, 삭제, migration은 승인된 design.md 이후에만 수행한다.
