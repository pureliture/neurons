승인 대상

  • Source of truth: requirements.md
  • Preview companion: requirements.html

요약

ingress-api는 SPRING_PROFILES_ACTIVE=api 단독으로 떠야 한다. target adapter가 없으면 앱을 죽이지 않고 /status에서 not_configured degraded 상태를 표현한다.

핵심 결정

api & !worker & !retired-index-bridge profile 전용 fallback RagTargetAdapter bean을 둔다. 실제 adapter profile 또는 worker profile이 켜지면 fallback은 비활성화된다.

검증

  • api profile status context smoke
  • /healthz와 /status contract test
  • 기존 retired-index-bridge와 worker 관련 test 유지
  • Java test와 boot jar build
