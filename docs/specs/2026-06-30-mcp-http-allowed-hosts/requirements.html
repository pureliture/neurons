MCP HTTP Allowed Hosts Requirements

Source of truth: requirements.md

Preview companion: requirements.html

승인 상태: 사용자가 requirements.md와 후속 design.md를 사전 승인함.

핵심 요구

  • neuron-knowledge mcp-http는 반복 가능한 --allowed-host를 지원한다.
  • MCP_HTTP_ALLOWED_HOSTS는 comma-separated 추가 Host/authority 목록을 지원한다.
  • Compose MCP service는 brain env file fallback을 보존하고, project env 값이 있을 때만 canonical env로 승격한다.
  • 추가 Host는 TransportSecuritySettings.allowed_hosts에 들어간다.
  • 추가 Host마다 https://<authority> origin도 함께 허용한다.
  • DNS rebinding 보호, loopback/tailnet/Pod IP bind guard, optional extra 구조는 유지한다.
  • 실제 운영 hostname과 private 값은 public repo 산출물에 기록하지 않는다.
